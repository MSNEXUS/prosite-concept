# ProSite — Construction Site Management Platform

A SaaS concept designed to digitise and streamline 
construction site operations for Indian SMEs.

## Overview
ProSite addresses a gap in the Indian construction market — 
the lack of affordable, practical site management software 
for small and medium contractors. This repo documents the 
concept, MVP scope, recommended tech stack, and product roadmap.

## Problem
Indian construction SMEs rely on manual processes — 
WhatsApp groups, paper logs, and spreadsheets — 
to manage labour, materials, timelines, and compliance. 
This leads to cost overruns, delays, and poor accountability.

## Proposed Solution
A lightweight, mobile-first SaaS platform covering:
- Daily site progress logging
- Labour attendance and payroll tracking
- Material inventory and procurement management
- Document storage (drawings, permits, safety records)
- Client-facing progress reports and photo updates
- Budget vs. actual cost tracking

| Feature | Priority |
|---------|----------|

| Labour attendance | P1 |

## Recommended Tech Stack
| Layer | Technology |
|-------|------------|
| Frontend | React Native (mobile-first) |
| Backend | Node.js + Express |
| Database | PostgreSQL |
| Storage | AWS S3 |
| Auth | Firebase Auth |
| Hosting | AWS / Vercel |

## Roadmap
- **Phase 1** — MVP: Site diary, labour, materials (3 months)
- **Phase 2** — Client portal and reporting (2 months)
- **Phase 3** — AI-based cost forecasting and delay prediction (ongoing)
